# Загрузчик картинок с котами

Загружает картинки кошек с текстом на Яндекс.Диск.

## Установка
1. Получите токен Яндекс.Диска: https://yandex.ru/dev/disk/poligon/
2. Вставьте свой токен в `main.py` вместо `"ваш_токен_здесь"`
3. Установите зависимости: `pip install requests`

## Использование
Запустите: `python main.py`
Введите текст для картинки

## Функции
- Получает картинки с API cataas.com
- Загружает на Яндекс.Диск
- Создает JSON файл с информацией
- Название папки = название вашей группы в Нетологии
